# LimitlessTask

Niestety z powodów prywatnych i zawodowych nie udało mi się znaleźć wystarczającej ilości czasu na wykonanie całego zadania
Wykorzystany język: kotlin
Wykorzystany framework: Spring Boost
Wykorzystany interface: CrudeRepository

Poprzez przesłanie request'a na metodę POST z textem jest on dodawany do kolekcji oraz zyskuje uuid.
Poprzez przesłanie request'a na metodę GET otrzymujemy wszystkie elementy zbioru
Poprzez przesłanie request'a na metodę DELETE z podanym textem oraz uuid elementu możemy usunąć element z kolejkcji.
